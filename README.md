# SpiralWeb

A Python visualization tool that generates geometric spiral patterns with radial lines.

## Overview

SpiralWeb creates an artistic spiral visualization by plotting an Archimedean spiral overlaid with radial lines radiating from the center. The result is a mesmerizing geometric pattern that expands outward.

## Technologies

- **Python 3** - Programming language
- **NumPy** - Numerical computing and array operations
- **Matplotlib** - Data visualization and plotting

## Features

- Generates a smooth spiral curve with configurable turns
- Overlays radial lines from the center to the spiral
- Customizable parameters:
  - `num_lines` (default: 1500) - Number of radial lines
  - `num_turns` (default: 150) - Number of spiral rotations
  - `num_points` (default: 10000) - Resolution of the spiral curve

## Installation

1. Ensure Python 3 is installed on your system
2. Install the required dependencies:

```bash
pip install numpy matplotlib
```

## Running

Simply execute the script from the command line:

```bash
python spiralweb.py
```

A window will open displaying the spiral pattern. Close the window to exit.

## Customization

Edit the parameters at the top of `spiralweb.py` to change the appearance:

```python
num_lines = 1500      # Increase for more density
num_turns = 150       # Increase for more spiral rotations
num_points = 10000    # Increase for smoother curves
```
